Filter WC Orders adds a new filter dropdown on the admin order screen.

== Description ==

It provides a way to admin for filtering woocommerce orders based on payment gateways and user types.

== Features ==

1. **Payment Gateway Based Filters**
Admin can view all orders processed through a specific payment gateway (Screenshot-1)

2. **User Type Based Filters**
Orders can be filtered also by Guest users and shop customers

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/filter-wc-orders` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the \' Plugins\' screen in WordPress

3. Go to the woocommerce orders page on the admin dashboard.

4. Select the option from the new dropdown and apply the filter.

== Frequently Asked Questions ==

= What types of filters are in this plugin? =
Admin and shop-managers can filter orders based on installed Payment Gateways and user types viz. guest and logged in.
