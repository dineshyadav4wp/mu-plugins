== Description ==

Provide options to set a dynamic or static password from settings.

Features:

Can add a password prefix.
Can select frequency as suffix.
Can add static password.
Can enable/disable the auto password update.

== Installation ==

1. Clone the repo as mu-plugins if it is not already there. Or copy just 'dyanmic-password-manager' folder and main file to mu-plugins.
2. Configure the plugin DPM settings
